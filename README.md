# Sentry Secure – AI Construction Site Monitoring System

## 📌 Overview
Sentry Secure is an AI-powered construction site monitoring system designed to improve worker safety by detecting safety violations and generating real-time alerts using computer vision and machine learning techniques.

The system uses YOLO-based object detection to identify safety compliance such as helmet usage, unsafe behavior, and hazardous conditions in construction environments.

---

## ⚙️ Features
- Real-time object detection using YOLO model
- Helmet / safety detection in construction sites
- Hazard detection and alert system
- Automated safety monitoring system
- Backend processing for inference

---

## 🧠 Tech Stack
- Python
- YOLO (Object Detection Model)
- OpenCV
- Machine Learning
- Flask / WSGI

---

## 📁 Project Structure
- app.py → Main application logic
- train.py → Model training script
- config.py → Configuration settings
- wsgi.py → Server entry point
- yolov8n.pt → Pretrained YOLO model

---

## 🚀 How to Run

### Step 1: Install dependencies
pip install -r requirements.txt

### Step 2: Move into project folder
cd sentry_secure

### Step 3: Run application
python app.py
